WEB DEVELOPMENT SOFTWARE INSTALLATION

The script, install.sh, included in this repo installs Mac-related configuration files and software required for any web development course you take with me. Before continuing, open a GitHub account at https://github.com/, regardless of whether you use Windows, Mac OS, or Linux.

Mac OS Users

Installation

  1. Open The Terminal
  2. cd into the root of this repo
  3. Type ./install.sh
