WEB DEVELOPMENT SOFTWARE INSTALLATION

The script included in this repo, install.sh, installs Mac-related configuration files and software required for any web development course you take with me, Roy Vanegas. Before continuing, do the following, regardless of whether you use Windows, Mac OS, or Linux.

  1. Open a GitHub account at https://github.com/
  2. Download GitHub Desktop at https://desktop.github.com/
  3. Download the Atom text editor at https://atom.io/
  4. Download the Sublime Text editor at https://www.sublimetext.com/3

Mac OS Users

Launch Max OS’s Terminal, navigate to this repo’s folder, then type ./install.sh
